	Устройство МУГС-03 (модуль управления гребне-смазывателем)
	Изделие предназначено для управления ЭПВ(вентиль электропневматический), включающими форсунки гребнесмазывателя вагонов метро в зависимости от режима движения ПС
Изделие содержит четыре канала управления внешней нагрузкой (ЭПВ), управляемых микроконтроллером, два входа для подключения внеш-них сигналов - «ЗАПРЕТ СМАЗКИ»
и «БАК», гироскоп, для определения криволинейного участка пути, акселерометр, для определения отрицательного ускорения (замедления подвижного состава), шину CAN
После подачи питания изделие проходит режим АВК, при успеш-ном завершении которого заработает зеленым цветом светодиод 
СОСТОЯНИЕ. Изделие сразу же переходит к основному режиму работы и автоматически определяет:
- скорость ПС;
- наличие криволинейного участка;
- наличие отрицательного ускорения (замедления);
- наличие или отсутствие внешних сигналов «Запрет смазки», «Датчик бака».
	Алгоритм работы изделие обеспечивает включение ЭПВ на время Tв, на заданном интервале L1, L2 или L3 в зависимости от скорости движения и
 наличия криволинейного участка пути. Данные о скорости поступают по шине CAN. Наличие криволинейного участка пути определяется встроенным гироскопом 
по превышению угловой скорости поворота вагона заданного порога ωпор=(0,6±0,2) град/с.
	Сигналы управления ЭПВ формируются микроконтроллером платы управления МУГС-02.
В состав каждого канала управления входит схема определения отказа - «ОБРЫВ», и схема определения отказа - «КЗ».
При выявлении этими схемами указанных типов отказов в цепях подключения ЭПВ,
формируются сигналы, которые поступают на микроконтроллер платы,
где они используются для блокировки формирования сигналов управления электронными ключами. 
В состав платы управления входит также постоянное запоминающее устройство ПЗУ, предназначенное для хранения информации с настройками

	Запрет выдачи управляющих сигналов на ЭПВ формируется либо при поступлении внешнего сигнала «Запрет смазки» либо при торможении подвижного
 состава с отрицательным ускорением (замедлением) выше порогового.Данные об ускорении берутся либо от внутреннего датчика – акселерометра,
 либо поступают из шины CAN. После поступления одного из вышеуказанных сигналов изделие работает по одному из двух алгоритмов:
- выключает ЭПВ на время поступления сигнала запрета, при этом не проис-ходит сброс счётчика пройденного пути, в момент
 поступления сигнала (форми-рование заданного интервала пути не прерывается);
- выключает ЭПВ на время поступления сигнала запрета, при этом происхо-дит сброс счётчика пройденного пути в момент поступления сигнала
 (формирование запрограммированного интервал пути прерывается и начинает формироваться с момента поступления сигнала запрета).

	В изделии предусмотрена СВК, работающая в двух режимах:
	-ПВК, включающегося только принудительным нажатием на кнопку 
КОНТРОЛЬ на лицевой панели  или по соответствующей команде по шине CAN при стоянке ПС;
	-АВК, включающегося автоматически, с подачей питания.

	ПВК запускается кратковременным нажатием кнопки КОНТРОЛЬ. Далее через заданное время T1 изделие:
	- три раза включает первый канал ЭПВ с заданными значениями времени включения 
	Tв и длительностью паузы между включениями ЭПВ Тп =4с;
	- выполняет тестовую проверку исправности внутренних датчиков ускорения (акселерометра)
 	и угловой скорости (гироскопа).
Результаты СВК передаются по шине CAN и индицируются свето-диодом СОСТОЯНИЕ
АВК запускается при подаче питания. Результаты тестирования ин-дицируются светодиодом СОСТОЯНИЕ и передаются по шине CAN.
В режиме АВК изделие обеспечивает автоматическое определение:
- неисправностей в шине CAN;
- неисправностей «ОБРЫВ» или «КЗ» в цепи управления ЭПВ во время движения ПС и отключение неисправного канала
Возобновление подачи смазочного материала на ЭПВ, в котором была выявлена неисправность, 
производиться только после успешно пройденного ПВК, либо перезапуска изделия.
